## Getting Started

To run the Todo App on your local machine, follow these steps:

1. Clone this repository:

```bash
  git clone https://github.com/aleyaktas/TodoApp.git
```

2. Open the project in Xcode:

```bash
  cd TodoApp
  open TodoApp.xcodeproj
```
